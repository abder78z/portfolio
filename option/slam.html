SLAM

La spécialité SLAM (Solutions Logicielles et Applications Métiers) forme des développeurs capables de concevoir, développer et maintenir des applications informatiques répondant aux besoins des entreprises. Elle est axée sur la programmation, les bases de données et le développement d’applications web et logicielles.




En BTS SIO option SLAM, les principaux langages qui peuvent être étudiés sont :

  • Python
  • PHP
  • Java
  • JavaScript
  • HTML / CSS
  • SQL (bases de données)
Selon les établissements, d’autres technologies ou frameworks peuvent aussi être abordés.
